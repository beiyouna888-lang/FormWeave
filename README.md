# FormWeave（形织）— 面向工程设计场景的智能设计助手

FormWeave 是一款大模型驱动的 AI CAD Agent，工程师通过自然语言即可生成三维参数化模型、工程图纸及设计交付物。

## 核心功能

- 语言驱动建模** — 自然语言描述 → 完整参数化三维模型
- **智能布局优化** — 给定空间约束，自动生成最优布局方案
- **参数化族管理与知识沉淀** — 模板抽取 + 设计规则图谱
- **设计审查助手** — 自动检测干涉/不合规项 + 修改建议

## 技术架构

```
用户输入层 → 意图理解层(LLM) → 几何推理层 → 约束求解层 → 渲染输出层
```

- **基座模型**：Llama / Qwen 系列工程设计微调
- **几何引擎**：OpenCASCADE (OCCT)
- **约束求解**：Z3 SMT + 自研规则引擎
- **CAD 适配**：FreeCAD API → SolidWorks / Autodesk 插件
